# final-web-project
Hello everybody who stumbles across this page. This project is my Final for my Intro to Web Design where I had
to create a site according to a customer's business philosophy, goals, and desires. The company "GreenTech Solutions"
is a company dedicated to providing businesses ways to be environmentally friendly. As, such the colors used in this site are
a series of greens and earthier tones and the design of some elements are meant to provoke the feeling of environmental friendlyness.

This site doesn't require anything specific in order to function, this site is able to stand on its own.

This site has so far been tested through Iphone14 and various types of Ipads' screen sizes with some changes to the navigation bar
depending on the size of the device the site is being viewed on.
